# Dataset (Beginner)

This folder is for sample legal text files.

## How it works in the app
- The backend loads **.txt** files from this folder.
- Each file is summarized using a HuggingFace summarization model.

## Expected file format
- Put your legal text in files like:
  - `dataset/sample_1.txt`
  - `dataset/sample_2.txt`

## Next steps for beginners
- Add more `.txt` files.
- Refresh the dataset route (see `/dataset_test`).
